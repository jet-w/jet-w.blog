## Classification
拦截器一共四类
1. 请求成功拦截
2. 请求失败拦截
3. 响应成功拦截
4. 响应失败拦截


## Example
``` js
instance.interceptors.request.use(
  config => {
    //请求拦截的作用
    //1. 修改config中的一些配置信息
    //2. 发送请求时在界面中显示图标
    //3. 某些网络请求（如登录），需添加特殊信息
    return config
  }, err => {
    
  }
)
instance.interceptors.response.use(
  res=> {
    return res.data;
  }, err => {
  }
)
```
