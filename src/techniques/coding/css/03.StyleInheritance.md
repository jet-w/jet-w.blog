***Definition:*** The style for one element, will be used on the descendants of the element.

``` html
<style>
p {
  color: red;
}
</style>

<p> 
  我是一个p元素
  <span> 我是p中的span元素</span>
</p>
```
::: warning
Not all the style will be inherited, such as background related, layout, etc.
:::
